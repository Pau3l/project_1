# PT Design - Payment Management System

A premium, high-performance dashboard for worker payment tracking and verification.

## 🚀 Key Features
- **Digital Signature**: In-app canvas for signature recording and viewing.
- **Workflow State Machine**: Robust status transitions (`Pending` → `Processing` → `Completed`).
- **Interactive Analytics**: Real-time volume trends and method distribution.
- **Data Export**: Support for CSV, Excel, JSON, and PDF.

## 🛠️ Tech Stack
- React 18.3, Vite 6, TypeScript
- Tailwind CSS 4.0, Framer Motion
- Radix UI Primitives, Lucide Icons, Recharts

## 📁 Project Structure
- `src/app/App.tsx`: Main application and workflow logic.
- `src/app/components/`: Modular UI components.
- `src/styles/`: Theme and design system.

## 🛠️ Development
```bash
npm install
npm run dev
```
